## Quick Start - Front-end

### 1.1. Instalando dependências

Clone o repositório na sua máquina

### 1.2. Vá a pasta e rode:

````
yarn
````
para instalar as dependências e libs


### 1.3. Iniciar a aplicação:

 Rode:

````
yarn dev
````
